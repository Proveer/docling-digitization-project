# Document Digitization Project

This project aims to digitize various unstructured document formats, including PDF, DOCX, PPTX, HTML, and images, using the `docling` library. The goal is to convert these documents into a structured format that can be used for further analysis or processing.

## Getting Started

### Prerequisites

* Python 3.8+
* An active Python virtual environment

### Installation

1. Clone this repository.
2. Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

1. Place the documents you want to digitize in the `data` directory.
2. Run the digitization script:

```bash
python src/digitizer.py
```

3. The digitized documents will be saved in the `output` directory.
